# Security Toolbox

This toolbox includes several updated tools for red team assessments and penetration testing running exclusively on docker


## Usage

```
cat 1.txt  | docker run -i toolbox hakrawler -subs
```

## Installation

```
docker build . -t toolbox:latest
```
